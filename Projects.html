Message Encryption Generator

Have you ever wondered about classical encryption methods and wanted to try it on your own messages?
Well one of the projects that I, along with 4 of my colleagues created was a program that randomly picks an encryption method for any message you want. Simply type all the messages you want, and when you no longer want to add inouts type 'Stop' or 'stop'. The program will end the user input and generate the encrypted version of your messages and even let you know which encryption method it used! The encryption methods used are, Substitution, Playfair, Transposition, Shift, Product, and RSA.

This document is an extensive report diving into the history of the Atari 2600, the architecture at that time, and how that architecture influenced upcoming architecture even till today.
